# README

* 概要  
　　情報共有掲示板です。（Twitterをベースにしています。）  
  
* 機能・技術一覧  
　・認証機能（メール認証）：　devise, sendgrid  
　・記事投稿、コメント機能：　Ajax  
　・ページネーション機能：　　kaminari  
　・画像アップロード機能：　　AWS S3  
　・フォロー・フォロワー機能  
  
* その他、使用している技術  
　・テスト：　　　　rspec  
　・データベース：　postgreSQL  
　・インフラ：　　　heroku  
　・CSS：　　　　　Bootstrap, FontAwesome  
　　　　　　　　　　レスポンシブデザイン
